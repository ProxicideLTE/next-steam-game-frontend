# Next Game Frontend

This repo contains the front-end layer for the next game website built using
React, Redux, and TailwindCSS.

## Requirements

- node v17-19
- npm v6-9
- Next Game backend repo installed
- MongoDB

## Installation

To install all dependencies for this project:

```bash
npm install
```
